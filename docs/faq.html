What does this tool do?

It reads the approximate waste-ink maintenance counters that some printers expose over USB, backs up the relevant printer memory, and can reset those counters after you have physically serviced the waste-ink pad/tank.

Does resetting fix a full waste-ink pad?

No. A reset clears only a software counter. The physical pad is still full. You must inspect, clean, replace the pad, or install/verify an external waste-ink tank first — otherwise ink can overflow inside the printer.

Is it safe?

The tool reads first, takes an automatic backup before any reset, and requires explicit confirmations. Writing to printer memory still carries some risk, so use it responsibly and only after physical service. It is provided “as is”, without warranty.

Which models are supported?

The L3250 / L3251 / L3253 / L3255 family is verified on hardware. Additional models are marked experimental (data not verified on hardware by this project). See the supported models list.

Why does Windows warn when I run it?

The executable is not code-signed, so SmartScreen may show “Windows protected your PC”. Click More info → Run anyway. You can also verify the SHA-256 against SHA256SUMS.txt, or build from source.

My antivirus flagged the .exe.

PyInstaller executables are sometimes false-flagged. Verify the published SHA-256, or build it yourself from source.

Does it collect my data?

No telemetry. Logs and backups are stored locally on your device. The local log records the model, counters, backup path, confirmation flags and errors — not serial numbers or personal data. See PRIVACY.md.

Does it work over Wi-Fi?

No — USB only.

It says my model isn’t in the database.

Only listed models are supported. You can help add and verify a model legitimately — see CONTRIBUTING.md.

The error didn’t clear after a reset.

Power-cycle the printer (off with its power button, wait a few seconds, on) to commit the change.
