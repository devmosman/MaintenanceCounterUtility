What it does

It can
  • Read approximate waste-ink maintenance counters over USB.
  • Read usage counters (printed pages, scans, cleaning cycles).
  • Back up the relevant printer memory before any change.
  • Reset the maintenance counters after you service the pad/tank.
It does not
  • Clean, empty, repair, or replace the waste-ink pad/tank.
  • Perform any physical maintenance.
  • Act as a way to avoid service requirements.
  • Guarantee compatibility, or provide legal advice.

Important — physical maintenance first

A counter reset clears only the software counter. The physical waste-ink pad is still full and must be inspected, cleaned, replaced, or redirected to an external tank — otherwise ink can overflow inside the printer. Use this only for legitimate owner maintenance, after physical service.

Supported models

Verified on hardware (June 2026): L3250 / L3251 / L3253 / L3255.
Experimental (unverified): additional compatible models — use extra caution.

See the full model list and data provenance.

Install & verify

  • Download the latest release, unzip, run MaintenanceCounterUtility.exe.
  • The app is unsigned: SmartScreen may warn → More info → Run anyway.
  • Verify integrity against SHA256SUMS.txt, or build it yourself from source.
  • Connect the printer by USB, accept the Terms on first run, then use the app.
  • After a reset, power-cycle the printer to commit the change.
